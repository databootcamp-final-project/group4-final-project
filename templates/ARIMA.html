In [21]:
diff = df["Close/Last"].diff().diff().dropna()

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(16, 4))

ax1.plot(diff)
ax1.set_title("After Second Difference")

plot_acf(diff, ax=ax2)

There is no big difference in the second correlation plot, compared to the first one, except for the large negative value at lag 2. This indicates that the series was probably over-differentiated, and the correct value for the "i" term in the ARIMA(p,i,q) model is 1.

Another way to get the correct value for the number of differences needed is by use of the pmdarima.ndiffs() function:

p is theNumber of lags to be used as predictors and represents the order of the Auto Regressive term of the model equation (the "p" in ARIMA(p,i,q). The partial autocorrelation is used for determining the "p" term.
